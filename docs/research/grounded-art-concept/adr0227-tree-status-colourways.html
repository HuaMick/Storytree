The tree carries its status again

The tree-spread made every island's central tree the baked autumn-tree hero. A baked hero's paint is fixed, so every tree read one brown — "this looks good but all the trees are brown?". The fix bakes that one authored silhouette in a colourway per status, so a tree tells you its story's state by its crown again — green when proven, red when failing — while staying the same tree.
